# selene-ui
TW Selene Fixed Asset React UI

##NPM

- Instalar node y nvm
```sh
$ brew install node
$ npm install -g nvm
```
Verificar la versión instalada
```sh
$ node --version
```

Descargar e instalar la versión de NVM
```sh
$ nvm download 4.4.4
$ nvm build 4.4.4
$ nvm install 4.4.4
```

Instalar las dependencias de nuestra aplicación

Dentro del directorio ```selene-ui``` ejecuta

```
npm install
npm start
```
